# Audiokiste

This is Audiokiste a fork of Audio Adventcalendar by Mischka.

Uses...

* JQ6500 mp3 module
* Arduino Nano instead of the UNO.

----

## Folder Structure on SD-Card:

  Folders 00 to 09 for keys 0 to 9.

  Files 001.mp3 to 999.mp3 for music files.
